# [오픈소스 경진대회] 전지적 독자 시점 공식 홍보 플랫폼 리뉴얼

## 1. 프로젝트 개요 및 개발 필요성 (Importance)

본 프로젝트는 글로벌 슈퍼 IP인 <전지적 독자 시점>의 신규 독자 유입을 확대하고, 분산된 팬덤 정보를 통합하기 위해 개발되었습니다. 기존 웹툰/소설 페이지의 한계를 넘어선 웹사이트 개발의 필요성은 다음과 같습니다.

### 개발 배경 및 웹페이지의 중요도
1. 진입 장벽 해소 (User Acquisition): 방대한 세계관과 10년이 넘는 서사 설정은 신규 독자에게 높은 진입 장벽으로 작용합니다. 이를 시각적 스토리텔링(Timeline)으로 요약하여 누구나 쉽게 세계관을 이해하도록 돕습니다.
2. 정보 접근성 개선 (User Convenience): 팝업스토어 일정, 피규어 예약, 이벤트 정보가 여러 SNS와 판매처에 흩어져 있습니다. 이를 한곳에서 제공하는 정보 허브를 구축하여 사용자의 불편을 해소합니다.
3. 팬덤 경험 강화 (Engagement): 텍스트로만 존재하는 복잡한 인물 관계를 시각화된 인터페이스로 구현하여, 독자들에게 새로운 콘텐츠 경험을 제공합니다.
