# 전지적 독자 시점 공식 팬 페이지

이 프로젝트는 웹툰 및 웹소설 **「전지적 독자 시점」**을 주제로 한
소개형 웹사이트입니다. 작품의 세계관, 팝업스토어, 굿즈, 이벤트 정보를
한눈에 확인할 수 있도록 구성되었습니다.

---

## 📌 프로젝트 목적
- 전지적 독자 시점을 처음 접하는 사람에게 작품을 소개
- 팬을 위한 팝업스토어 및 굿즈 정보 정리
- HTML/CSS 기반 웹사이트 제작 실습

---

## 🛠 사용 기술
- HTML5
- CSS3
- GitHub Pages
- Git & GitHub 협업

---


## 📂 페이지 구성
- `index.html` : 메인 메뉴 페이지
- `popup.html` : 팝업스토어 & 굿즈 안내
- `deeppopup.html` : 팝업스토어 상세 소개
- `deepgoods.html` : 굿즈 상세 안내
- `deepevent.html` : 이벤트 상세 안내
